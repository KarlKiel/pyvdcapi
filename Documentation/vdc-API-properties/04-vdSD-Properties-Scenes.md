### 4.1.4 Scenes

- This property is available on devices having at least one output with standard behaviour defined, such as light, or shadow. Input-only devices do not support this property.

| Property Name | acc | Type / Range | Description |
|---|---|---|---|
| scenes | r/w | optional list of property elements | The scene table of the device<br>• represented as a list of property elements, one for each scene.<br>• The property elements are named by scene number, starting with "0".<br><br>See 4.10 Scene Descriptions for details |
